# Projeto de Estudo - Banco de Dados MySQL 🚗

Projeto desenvolvido como prática no curso "Aprenda SQL - MySQL".

O curso aborda conteúdos do nível básico ao avançado.  
Atualmente possuo domínio dos principais conceitos e comandos mais utilizados no dia a dia, como criação de tabelas, relacionamentos, chaves primárias e estrangeiras, inserts e consultas.

## 🎯 Objetivo

Simular um pequeno sistema de concessionária para praticar modelagem e estruturação de banco de dados relacional.

## 🗂 Estrutura do Projeto

- carros_marcas.sql → Cadastro de marcas
- carros_clientes.sql → Cadastro de clientes
- carros_inventario.sql → Cadastro de veículos disponíveis

## 🔗 Conceitos aplicados

- Chave primária
- Chave estrangeira
- Relacionamento entre tabelas
- Normalização básica
- Organização de scripts SQL

## 🛠 Tecnologias utilizadas

- MySQL
- MySQL Workbench

## 📌 Observação

Este repositório faz parte da minha evolução nos estudos em Banco de Dados e continuará sendo atualizado conforme avanço nos conteúdos.
