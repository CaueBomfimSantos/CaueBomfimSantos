## 💻 Estudos em JavaScript – Lógica de Programação

Este repositório reúne meus exercícios de lógica de programação utilizando JavaScript.

Sou iniciante na linguagem e estou utilizando este espaço para praticar os fundamentos e registrar minha evolução.

Os exercícios são baseados na lista de Algoritmos do professor Gustavo Guanabara.

## 🎯 Objetivo

Praticar os fundamentos da programação:

Estruturas condicionais (if, else)

Estruturas de repetição (while, for)

Arrays

Funções

Operações matemáticas

Lógica e resolução de problemas

O foco não é interface ou design, mas sim desenvolver raciocínio lógico.

## 🛠 Tecnologias

JavaScript

Node.js

Utilizo o Node.js para executar os arquivos .js pelo terminal, sem depender do navegador.

## 📂 Organização

Os arquivos estão separados por tema (sequências, condições, repetições, vetores e funções), conforme avanço nos estudos.

📌 Sobre este projeto

Este repositório faz parte da minha rotina de estudos e será atualizado conforme eu evoluir na linguagem.
