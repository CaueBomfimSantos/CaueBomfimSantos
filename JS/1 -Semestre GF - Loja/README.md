## 🛒 Projeto Loja Online

Projeto desenvolvido no 1º semestre da graduação em Banco de Dados.

Foi uma atividade prática para aprender os conceitos básicos de desenvolvimento web utilizando HTML, CSS, Bootstrap e JavaScript.

## 🎯 Objetivo

Criar uma estrutura simples de loja online para praticar:

HTML (estrutura das páginas)

CSS (estilização)

Bootstrap (layout e componentes)

JavaScript (interações básicas)

## 🛠 Tecnologias utilizadas

HTML

CSS

Bootstrap

JavaScript

## 📂 Organização

O projeto está separado em pastas para organizar melhor os arquivos:

css → estilos

js → scripts

img → imagens

arquivos .html → páginas do sistema

## 📌 Observação

Este projeto faz parte do meu início na programação e representa meu aprendizado durante o primeiro semestre da faculdade.
